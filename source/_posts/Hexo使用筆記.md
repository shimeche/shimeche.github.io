### 發表文章

``` bash
$ hexo n [type] [title]
or
$ hexo new [type] [title]
```
[type]可以是post, page or draft, 預設是post

### 發表文章 Example
``` bash
$ hexo n 這是一篇文章
or
$ hexo new 這是一篇文章
```

### 編譯
``` bash
$ hexo g
or
$ hexo generate
```

### 開啟自建伺服器進行預覽
``` bash
$ hexo s
or
$ hexo server
```
照著訊息指示瀏覽網址就可以了
